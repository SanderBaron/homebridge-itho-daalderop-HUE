Turbo modus

De Turbo-schakelaar in HomeKit draait de ventilator op maximaal en keert na deze tijd automatisch terug.

Vochtigheidsautomaat

Boost triggert ook bij stijging van X% binnen Y seconden. Stel in op 0 om uit te schakelen. Itho spec: 3% / 24s.

Drie zones: onder minimum-drempel → laag, tussen drempels → auto, boven absolute drempel → hoog.

Tijdschema-overrides

Spiegelverwarming

Stuurt een Hue-lamp (spiegelverwarming) aan op basis van de externe RFT-RV vochtigheid.

Annuleerdrempel: alleen gebruikt tijdens de vertraging na de fan-boost — is de RFT-RV dan alweer onder deze waarde, dan gaat de spiegel niet aan. Verlengt de brandtijd niet. Leeg laten = uit.

Spiegel triggert ook bij stijging van X% binnen Y seconden — ook als de inschakeldrempel nog niet bereikt is. Stel in op 0 om uit te schakelen. Itho spec: 3% / 24s.

Toilet-ventilatie

Detecteert het toiletlicht via een Hue input-only schakelaar en draait de CVE op maximale stand.

Dagelijkse reset

Elke nacht stuurt de plugin het commando 'automatisch' — veiligheidsnet zodat de CVE nooit per ongeluk in een handmatige stand blijft staan.
